Project Overview

1. Project Overview

This project is a text-based RPG game where players battle against various monsters in a fantasy world, such as goblins, orcs and dragons.

The game features multi-stage design, an action selection system (attack, defense, evasion), as well as difficulty modes and hidden mechanisms (such as silver swords and life potions).

The project outcome is a playable and strategic single-player turn-based game that has been successfully tested and run on local terminals.

2. End Users

a. Who are they:

The end users are beginner programmers who are interested in Python programming and turn-based combat games, as well as students who wish to learn conditional judgment loops and variables through simple game logic.

b. Requirements:

Desired is a game project with high playability and easy-to-understand features.

I hope to learn the basic logical structure of Python programming through this project.

Like terminal games that are simple, highly interactive, and have intuitive operation effects.
