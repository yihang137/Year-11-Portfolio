Blender Project

Project Overview

In this project, I used Blender to complete a simple 3D modeling exercise. My goal was to master the basic operations of Blender, including modeling, the use of editing mode, and the addition of simple materials. Eventually, I created a model set that is suitable for character models in a game environment.

During this process, I learned:

  • How to add and modify basic geometric shapes (such as cubes and cylinders) using the editing mode
  • How to apply various basic operations

2. End Users

a. Who are they:

My target users might be younger children who enjoy games and students who are new to modeling. They might prefer simpler, fantasy character modeling, or need practice materials when learning the 3D workflow.

b. Requirements:

It is desired that the model be simple and easy to understand, with uncomplicated materials, and be convenient for modification or further creative use.

Detailed Specifications

Requirements Specification Description
Number of Polygons Each model should have no more than 10 polygons
Size Ratio Height ratio of head, body and feet is approximately 1:2:0.5
Material Use basic color materials (without textures)

5. Development Log

  • Day 1: Understand the project goals and the Blender interface, import reference images
  • Day 2: Create the basic cube and use the extrusion and scaling operations to build the main body of the model
  • Day 3: Complete the structure of the model
